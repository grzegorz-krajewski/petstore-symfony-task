# Petstore Api

Zadanie rekrutacyjne polegające na przygotowaniu prostej aplikacji w Symfony, komunikującej się z zewnętrznym REST API.

## Zakres
Aplikacja umożliwia:
- dodawanie elementów w zasobie /pet,
- pobieranie elementu po ID,
- edycję elementu,
- usuwanie elementu,
- obsługę błędów,
- prosty interfejs formularzy.

## Założenia
Frontend został ograniczony do prostego interfejsu formularzy, ponieważ główny nacisk w zadaniu położony jest na backend i integrację z API.
